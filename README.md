# Sess[a]Pool Lang

[https://github.com/mweppler/sess-a-pool]

-

## DESCRIPTION

wikipedia: "A cesspit, or cesspool, is a pit, conservancy tank or covered cistern which can be used to dispose of urine and feces..."

That pretty much sums things up...

-

## Usage

```
bundle install

# build the lexer
rake lexer

# build the parser
rake parser

# run tests
rake spec
```

-

## REQUIREMENTS

ruby 2.3.0
